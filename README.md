# Basics-of-cybersecurity

## Vulnerability Scanner

Код реалізує простий сканер відомих вразливостей програмного забезпечення з використанням NVD від NIST.

Скрипт:
- перевіряє обрані програми;
- отримує інформацію про CVE;
- зберігає результати у CSV;
- виводить статистику в термінал;
- будує візуалізацію розподілу вразливостей за рівнем небезпеки.
